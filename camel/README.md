# 🐪 CamelCabal Terminal v1.0

### The Official DN404 Scarcity & Mint Command Center

CamelCabal Terminal is a high-performance, real-time monitoring dashboard designed for the **CamelCabal DN404 ecosystem**. It provides holders and traders with immersive, high-utility data visualization of token-to-NFT solidification.

![Dashboard Preview](https://cdn.dexscreener.com/cms/images/8HLGxaEdQowSDERh?width=1200&height=630&quality=95&format=auto)

---

## 🎨 Key Aesthetics
*   **Clandestine UI:** Inspired by vintage terminals and Silk Road legacy.
*   **CRT Engine:** Built-in scanline overlays and phosphor-glow effects.
*   **Modern Typography:** Powered by **JetBrains Mono** for maximum legibility.

## 🧠 Core Features
*   **Live Mint/Burn Tracker:** Real-time WebSocket monitoring of the Ethereum blockchain.
*   **Global Scarcity Score:** A proprietary health metric (0-100) based on supply absorption and market premiums.
*   **Floor Spread Analyzer:** Real-time comparison between the OpenSea Floor and Implied Token Floor.
*   **Persistent Caching:** Uses `localStorage` to save blockchain events for instant loading on repeat visits.
*   **Cabal Alerts:** Live "Terminal Popups" for every on-chain event.
*   **Fully Responsive:** Optimized for both Desktop and Mobile tracking.

## ⚡ Technical Stack
*   **Ethers.js:** For direct blockchain interaction.
*   **Chart.js:** For supply distribution and activity analytics.
*   **Multi-Proxy Fallback:** Redundant price feed architecture (DexScreener & OpenSea).
*   **Pure Static Build:** No backend required. 100% client-side for maximum privacy and decentralization.

## 🚀 Deployment
This project is a static site. To deploy:
1. Upload the files to **Vercel**, **Netlify**, or **GitHub Pages**.
2. No build step required.
